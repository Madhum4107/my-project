# CPU Memory Predictor

This project predicts CPU and memory usage using Machine Learning.

## Features
- Real-time CPU and memory monitoring
- Future prediction using Linear Regression
- Alerts for high usage
- Graph visualization

## Technologies Used
- Python
- psutil
- pandas
- scikit-learn
- matplotlib

## How to Run
1. Install required libraries
2. Run the Python file
3. View predictions and graph output
   ## Output
The program collects CPU and memory data, predicts future usage, and displays alerts and graphs
